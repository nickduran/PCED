How to Use This Flowchart

  • Start at the top after reading all patient messages in the thread
  • Follow the arrows based on yes/no answers to each question
  • Remember: Focus on treatment/management options, not purely diagnostic speculation
  • If multiple patient messages: Code the strongest opportunity created
